Hospital-Admissions-Analyses-

--Overview
SQL analysis of a synthetic healthcare admissions dataset using Google BigQuery.

--Dataset
50,000 synthetic hospital admission records including demographics, medical conditions, treatments, and billing information.

--Tools
- SQL
- Google BigQuery

--Key Analyses
- Hospital stay patterns
- Medical conditions by gender
- Medication usage by condition
- Hospital-level condition analysis
- Test results by admission type

--Data Cleaning
Hospital naming inconsistencies were resolved by extracting the core
hospital identifier and standardising patient names.

--Repository Structure
Hospital Data SQL.sql – SQL queries used in the project
insights.md – key findings from the analysis
